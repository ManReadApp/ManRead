# ManRead

A mangareader completly written in rust
## Preview 
### Login/SignUp/Forgot Password
![Auth GIF](assets/auth.gif)
## Download
```sh
git clone https://github.com/ManReadApp/ManRead
cd ManRead
chmod 700 ./download.sh
./download.sh
```

## Run Server
```sh
cd api
cargo r --release
```

#### Run Client
```sh
cd app
cargo r --release
```
